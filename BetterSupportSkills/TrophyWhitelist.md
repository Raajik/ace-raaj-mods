## Notes

- **Default behavior:** All checked items above will stack 2-5 per TrophyDrops proc when Assess Creature is trained
- **Potions:** Stack 2-3 per proc when Alchemy is trained (separate from trophy stacks)
- **Extra rolls:** Based on buffed skill / 150, capped at 3 rolls
- **To disable an item:** Remove the `✅` or delete the row entirely
- **To add an item:** Add a new row with the item name and WCID
- **To change stack size:** Let me know the desired min/max per item
